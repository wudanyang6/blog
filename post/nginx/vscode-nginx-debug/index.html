1 通过源码安装 nginx

1.1 下载源码

1

git clone https://github.com/nginx/nginx

1.2 编译并安装 nginx

修改 nginx 源码中的配置文件：auto/cc/conf

将第 12 行修改一下

ngx_compile_opt="-c" 改成： ngx_compile_opt="-c -g"

-g 选项会让编译器产生调试信息

执行配置并安装：

1
2

cd nginx
auto/configure

默认被安装在 /usr/local/nginx ，可以通过 --prefix 选项修改

继续执行 make && make install

到这里，nginx 就安装完毕了

1.3 启动 nginx

1

/usr/local/nginx

默认监听本机 80 端口

完成设置后会出现下面的图片：

Welcome to nginx!
