using namespace std;

int main() {
    cout << "hello, world" << endl;
    return 0;
}

会经过4个步骤

  • 预处理
  • 编译
  • 汇编
  • 链接

预处理

作用：宏替换（展开文本，include 文本替换），删除空格等文本处理

相关语句 -E 仅生成 .i 预处理后的文件

编译

-S 仅生成 .s 编译后 .s 后缀文件

将c代码解析后生成汇编语句
